# Support

## Scope

- GitHub Issues are welcome for bugs, documentation fixes, and questions.
- External pull requests are **not accepted** (see `CONTRIBUTING.md`).

## Before opening an issue

Please include:

- the release tag you are using (example: `v3.431`),
- broker and symbol(s),
- timeframe(s),
- whether you are running live or in Strategy Tester,
- relevant MT5 logs (especially messages starting with `[INPUT-CONTRACT]`),
- if applicable: your `.set` preset file and the steps to reproduce.
