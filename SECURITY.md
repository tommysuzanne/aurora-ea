# Security policy

## Reporting a vulnerability

Please **do not** open a public GitHub Issue for security vulnerabilities.

Instead, report privately by email:

- `hello@tommysuzanne.com`

Recommended subject: `[SECURITY] Aurora EA`

Include (if possible):

- a short description of the issue and impact,
- steps to reproduce (or PoC),
- affected version tag (example: `v3.431`) and environment details.

## Disclosure

We will acknowledge receipt and coordinate a responsible disclosure timeline.
