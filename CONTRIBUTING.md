# Contributing (public mirror)

This GitHub repository is a **public mirror** of Aurora EA.

## Pull requests

External pull requests are **not accepted** at this time.

## Issues

Issues are welcome for:

- bug reports,
- documentation improvements,
- questions (please include context and reproducible steps).

Please follow the support checklist in `SUPPORT.md`.
