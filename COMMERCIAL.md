# Licence commerciale (Aurora EA)

## Objectif

Ce dépôt est distribué sous une licence **gratuite pour usage personnel** (voir `LICENSE`).
Toute **exploitation commerciale** d’Aurora EA nécessite une **licence commerciale** distincte.

## Périmètre “commercial” (exemples)

Sans s’y limiter, est considéré comme usage commercial :

- usage par une entreprise / personne morale ;
- usage dans le cadre d’une activité professionnelle (salariée ou indépendante) ;
- prestation, conseil, formation payante, intégration, ou maintenance pour un client ;
- fourniture de signaux, copie de trades, gestion de comptes tiers ;
- mise à disposition “as-a-service” (SaaS), hébergement, accès distant, API, bot partagé ;
- revente ou redistribution comme produit.

## Modèle de rémunération (principe)

Le modèle visé est une **redevance liée à la performance/profit** réalisé grâce à l’EA, définie contractuellement
avec l’entreprise (définition du “profit”, périmètre, reporting, droit d’audit, modalités de paiement).

## Process (high-level)

1) Premier échange (périmètre, usage, environnement, risques).
2) NDA (si nécessaire).
3) Contrat de licence commerciale (conditions, redevance, reporting, audit).
4) Onboarding (procédure d’installation, compilation, validation, go-live).
