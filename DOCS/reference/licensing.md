# Licence & usage (Aurora EA)

## Objectif

Décrire le modèle de licence d’Aurora EA :

- **usage personnel** : gratuit, code visible, modifiable ;
- **usage commercial** : payant, via contrat/licence commerciale.

## Sources

- `LICENSE`
- `COMMERCIAL.md`

## Résumé

- Vous pouvez utiliser/modifier Aurora EA **pour un usage personnel** (personne physique), y compris pour trader pour
  votre propre compte.
- Toute utilisation en entreprise, prestation, “as-a-service”, ou pour des tiers est **interdite** sans licence
  commerciale.

Commercial : `hello@tommysuzanne.com`

## Définitions (simplifiées)

- **Usage personnel** : usage par une personne physique pour son compte (même si cela génère un profit), hors cadre
  professionnel, et sans fournir un service à des tiers.
- **Usage commercial** : tout usage hors usage personnel, notamment usage par une entreprise/personne morale, ou usage
  dans le cadre d’une activité professionnelle.

## FAQ

### “Je suis un particulier et je gagne de l’argent avec l’EA, c’est autorisé ?”

Oui, tant que c’est **pour votre compte** et hors usage professionnel/prestation.

### “Je suis auto-entrepreneur / consultant, puis-je l’utiliser pour mes clients ?”

Non. C’est un **usage commercial** → licence commerciale requise.

### “Une entreprise peut-elle l’utiliser pour son trading interne ?”

Non via la licence gratuite. Licence commerciale requise.

### “Puis-je modifier le code ?”

Oui pour usage personnel. Pour usage commercial, cela nécessite une licence commerciale.

### “Puis-je redistribuer une version modifiée ?”

Oui uniquement à d’autres personnes physiques pour leur usage personnel, en conservant les notices et la licence.
